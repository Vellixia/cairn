# Cairn v0.8.0 — Implementation Plan

> **Goal:** Scoped memory (Session / Project / Global), SurrealDB replacing HelixDB+MinIO,
> auto project detection, LLM intelligence layer, RAG document store, session TTL + cron.

- **What:** what you are building
- **Why:** why it is needed
- **Files:** exact files to create or edit

- **Done when:** specific verifiable condition

Work the sprints in order (S1 → S7). Each sprint produces a working, passing build.
Never start S2 before S1 compiles. Run `cargo check --workspace` after every sprint.

```rust
use surrealdb::{Surreal, engine::any::Any};
use crate::StoreBackend;

pub struct SurrealStore {
    db: Surreal<Any>,
}

impl SurrealStore {
    /// Connect and apply schema. Called once at server startup.
    pub async fn open(url: &str, user: &str, pass: &str) -> Result<Self> {
        let db = surrealdb::engine::any::connect(url).await?;
        db.signin(surrealdb::opt::auth::Root { username: user, password: pass }).await?;
        db.use_ns("cairn").use_db("main").await?;
        db.query(include_str!("schema.surql")).await?.check()?;
        Ok(Self { db })
    }
}

impl StoreBackend for SurrealStore {
    // Implement each method by translating the old HelixDB calls to SurrealQL.
    // The trait method signatures do NOT change — only the implementation body changes.
}
```

For each `StoreBackend` method, map the HelixDB operation to SurrealDB:
- `store(memory)` → `db.create("memory").content(record).await?`
- `get(id)` → `db.select(("memory", id)).await?`
- `update(id, patch)` → `db.update(("memory", id)).merge(patch).await?`
- `delete(id)` → `db.delete(("memory", id)).await?`
- `list(filter)` → `db.query("SELECT * FROM memory WHERE ...").await?`
- `search_bm25(q)` → `db.query("SELECT *, search::score(1) AS score FROM memory WHERE content @1@ $q ORDER BY score DESC LIMIT $n").bind(("q", q)).await?`
- `search_vector(embedding, n)` → `db.query("SELECT *, vector::similarity::cosine(embedding, $vec) AS score FROM memory ORDER BY score DESC LIMIT $n").bind(("vec", embedding)).await?`

**HNSW index:** Create it after the first embedding is stored (we need the dimension):
```rust
pub async fn ensure_hnsw_index(&self, dim: usize) -> Result<()> {
    self.db.query(format!(
        "DEFINE INDEX IF NOT EXISTS idx_embedding ON memory \
         FIELDS embedding HNSW DIMENSION {dim} DIST COSINE TYPE F32;"
    )).await?.check()?;
    // Same for chunk table
    self.db.query(format!(
        "DEFINE INDEX IF NOT EXISTS idx_chunk_emb ON chunk \
         FIELDS embedding HNSW DIMENSION {dim} DIST COSINE TYPE F32;"
    )).await?.check()?;
    Ok(())
}
```

### S2.3 — Scope middleware in `cairn-api`

**File:** `crates/cairn-api/src/middleware/scope.rs` (new file)

This Axum middleware reads the incoming HTTP headers and injects a `ScopeCtx` into
request extensions. All handlers that need scope context pull it from extensions.

```rust
#[derive(Clone, Debug)]
pub struct ScopeCtx {
    pub project_id: Option<String>,
    pub session_id: Option<String>,
}

pub async fn scope_middleware(
    headers: HeaderMap,
    mut req: Request,
    next: Next,
) -> Response {
    let project_id = headers
        .get("X-Cairn-Project")
        .and_then(|v| v.to_str().ok())
        .map(str::to_string);
    let session_id = headers
        .get("X-Cairn-Session")
        .and_then(|v| v.to_str().ok())
        .map(str::to_string);
    req.extensions_mut().insert(ScopeCtx { project_id, session_id });
    next.run(req).await
}
```

Wire it into the Axum router in `crates/cairn-api/src/lib.rs`:
```rust
use axum::middleware;
use crate::middleware::scope::scope_middleware;

let app = Router::new()
    // ... existing routes ...
    .layer(middleware::from_fn(scope_middleware));
```

### S2.4 — Scoped recall in `cairn-memory`

**File:** `crates/cairn-memory/src/lib.rs` — function `recall()` (and `hybrid_search()`)

Replace the current unbounded query with a scope-aware WHERE clause:

```rust
/// Build the SurrealQL WHERE predicate for scope-aware reads.
/// Always includes: all Global + current Project + current Session.
fn scope_filter(ctx: &ScopeCtx) -> String {
    let mut parts = vec!["scope_type = 'global'".to_string()];
    if let Some(pid) = &ctx.project_id {
        parts.push(format!("(scope_type = 'project' AND scope_id = '{pid}')"));
    }
    if let Some(sid) = &ctx.session_id {
        parts.push(format!("(scope_type = 'session' AND scope_id = '{sid}')"));
    }
    format!("({})", parts.join(" OR "))
}
```

Inject into BM25 query:
```sql
SELECT *, search::score(1) AS score
FROM memory
WHERE content @1@ $q AND {scope_filter}
ORDER BY score DESC LIMIT $n
```

Inject into vector query:
```sql
SELECT *, vector::similarity::cosine(embedding, $vec) AS score
FROM memory
WHERE {scope_filter}
ORDER BY score DESC LIMIT $n
```

**New helper function in `hook.rs`:**
```rust
fn detect_project() -> (Option<String>, String) {
    // Priority 1: CAIRN_PROJECT env var (explicit override)
    if let Ok(name) = std::env::var("CAIRN_PROJECT") {
        if !name.trim().is_empty() {
            let id = project_hash(&name);
            return (Some(id), name);
        }
    }

    // Priority 2: git rev-parse --show-toplevel
    if let Ok(out) = std::process::Command::new("git")
        .args(["rev-parse", "--show-toplevel"])
        .output()
    {
        if out.status.success() {
            let path = String::from_utf8_lossy(&out.stdout).trim().to_string();
            let canonical = std::fs::canonicalize(&path)
                .unwrap_or_else(|_| std::path::PathBuf::from(&path));
            let name = canonical.file_name()
                .map(|n| n.to_string_lossy().into_owned())
                .unwrap_or_else(|| "unknown".to_string());
            let id = project_hash(&canonical.to_string_lossy());
            return (Some(id), name);
        }
    }

    // Priority 3: $PWD basename
    if let Ok(cwd) = std::env::current_dir() {
        let name = cwd.file_name()
            .map(|n| n.to_string_lossy().into_owned())
            .unwrap_or_else(|| "unknown".to_string());
        let id = project_hash(&cwd.to_string_lossy());
        return (Some(id), name);
    }

    (None, "unknown".to_string())
}

fn project_hash(path: &str) -> String {
    use sha2::{Sha256, Digest};
    let hash = Sha256::digest(path.as_bytes());
    hex::encode(&hash[..8])  // 16 hex chars = 64 bits, collision-resistant enough
}

fn current_dir_str() -> String {
    std::env::current_dir()
        .ok()
        .map(|p| p.to_string_lossy().into_owned())
        .unwrap_or_default()
}
```

```rust
pub async fn consolidate_stale_sessions(
    store: &dyn StoreBackend,
    ttl_days: u32,
    scope_default: &str,
    llm: Option<&LlmClient>,
) -> Result<usize> {
    if ttl_days == 0 { return Ok(0); }

    let cutoff = Utc::now() - Duration::days(ttl_days as i64);
    let stale = store.query(
        "SELECT * FROM session WHERE ended_at IS NULL AND started_at < $cutoff"
    ).bind(("cutoff", cutoff)).await?;

    let mut count = 0;
    for session in stale {
        // Step 1: working → episodic
        store.query(
            "UPDATE memory SET tier = 'episodic' \
             WHERE scope_type = 'session' AND scope_id = $sid AND tier = 'working'"
        ).bind(("sid", &session.id)).await?;

        // Step 2: session scope → project or global
        let target_scope = if session.project_hash.is_some() { "project" } else { scope_default };
        let target_id    = session.project_hash.as_deref();
        store.query(
            "UPDATE memory SET scope_type = $ts, scope_id = $tid \
             WHERE scope_type = 'session' AND scope_id = $sid"
        )
        .bind(("ts", target_scope))
        .bind(("tid", target_id))
        .bind(("sid", &session.id))
        .await?;

        // Step 3: LLM session summary (only if LLM enabled)
        if let Some(llm) = llm {
            let mems: Vec<Memory> = store.query(
                "SELECT content FROM memory WHERE scope_id = $sid"
            ).bind(("sid", &session.id)).await?;
            if !mems.is_empty() {
                let summary = llm.summarize_session(&mems).await?;
                store.create_memory(NewMemory {
                    content:    summary,
                    kind:       "note".into(),
                    tier:       "episodic".into(),
                    scope_type: ScopeType::Project,
                    scope_id:   session.project_hash.clone(),
                    importance: 0.6,
                    ..Default::default()
                }).await?;
            }
        }

        // Step 4: mark ended
        store.query(
            "UPDATE session SET ended_at = $now WHERE id = $sid"
        ).bind(("now", Utc::now())).bind(("sid", &session.id)).await?;

        count += 1;
    }
    Ok(count)
}
```

```rust
pub async fn detect_contradictions(
    store: &dyn StoreBackend,
    llm: &LlmClient,
) -> Result<usize> {
    let new_mems: Vec<Memory> = store.query(
        "SELECT * FROM memory WHERE created_at > $since AND suspicious = false LIMIT 50"
    ).bind(("since", Utc::now() - Duration::hours(24))).await?;

    let mut flagged = 0;
    for mem in new_mems {
        // Find top-10 semantically similar memories
        let similar: Vec<Memory> = store.search_vector(&mem.embedding, 10).await?;
        if similar.is_empty() { continue; }

        let candidates_str: Vec<String> = similar.iter().map(|m| {
            format!("ID: {} — {}", m.id, m.content)
        }).collect();

        let prompt = format!(
            "Does this statement contradict any of the candidates? \
             Statement: {}\n\nCandidates:\n{}\n\n\
             Return JSON: [{{\"id\": \"...\", \"contradicts\": true/false}}] for each candidate.",
            mem.content,
            candidates_str.join("\n")
        );

        if let Ok(json_str) = llm.complete(&prompt).await {
            if let Ok(results) = serde_json::from_str::<Vec<ContradictionResult>>(&json_str) {
                for r in results {
                    if r.contradicts {
                        // RELATE memory:mem_id ->contradicts-> memory:r.id
                        store.query(
                            "RELATE memory:$a ->contradicts-> memory:$b"
                        ).bind(("a", &mem.id)).bind(("b", &r.id)).await?;
                        store.query("UPDATE memory SET suspicious = true WHERE id = $id")
                            .bind(("id", &mem.id)).await?;
                        flagged += 1;
                    }
                }
            }
        }
    }
    Ok(flagged)
}
```

### S5.5 — Smart promotion scoring pipeline

**File:** `crates/cairn-memory/src/llm_intelligence.rs`

This runs inside the `llm-intelligence` cron job. Scores each `Project`-scoped
memory at `Semantic` or `Procedural` tier.

```rust
pub async fn score_promotion_candidates(
    store: &dyn StoreBackend,
    llm: &LlmClient,
    global_centroid: &[f32],  // precomputed by global-centroid job
) -> Result<Vec<PromotionCandidate>> {
    let candidates: Vec<Memory> = store.query(
        "SELECT * FROM memory \
         WHERE scope_type = 'project' \
         AND tier IN ['semantic', 'procedural'] \
         AND promo_locked = false \
         AND promo_score < 0.90"
    ).await?;

    let mut scored = Vec::new();
    for mem in candidates {
        // Step 1: Hard blocklist
        if contains_sensitive(&mem.content) { continue; }

        // Step 2: Kind prior
        let kind_prior = kind_prior_score(&mem.kind);
        if kind_prior == 0.0 { continue; }

        // Step 3: Cross-project signal
        let cross_count: i64 = store.query(
            "SELECT count() FROM access_log \
             WHERE memory_id = $id AND project_id != $pid \
             AND ts > $cutoff GROUP ALL"
        ).bind(("id", &mem.id))
         .bind(("pid", mem.scope_id.as_deref().unwrap_or("")))
         .bind(("cutoff", Utc::now() - Duration::days(30)))
         .await?;
        let cross_score = (cross_count as f32 / 3.0).min(1.0);

        // Step 4: Embedding universality
        let univ_score = cosine_similarity(&mem.embedding, global_centroid);

        // Step 5: Fast score
        let fast_score = kind_prior * 0.2 + cross_score * 0.5 + univ_score * 0.3;

        // Step 6: LLM judgment only in borderline range
        let final_score = if fast_score >= 0.40 && fast_score <= 0.85 {
            let prompt = format!(
                "Is this knowledge project-specific or universally applicable to any project? \
                 Score from 0.0 (project-only) to 1.0 (universal). Reply with just the number.\n\n{}",
                mem.content
            );
            if let Ok(s) = llm.complete(&prompt).await {
                let llm_score = s.trim().parse::<f32>().unwrap_or(fast_score);
                fast_score * 0.5 + llm_score * 0.5
            } else { fast_score }
        } else { fast_score };

        // Step 7: Update DB score
        store.query("UPDATE memory SET promo_score = $s WHERE id = $id")
            .bind(("s", final_score)).bind(("id", &mem.id)).await?;

        // Auto-promote if very high
        if final_score > 0.90 {
            store.query(
                "UPDATE memory SET scope_type = 'global', scope_id = NONE WHERE id = $id"
            ).bind(("id", &mem.id)).await?;
        }

        scored.push(PromotionCandidate { memory: mem, score: final_score });
    }
    Ok(scored)
}

fn contains_sensitive(text: &str) -> bool {
    // Regex for IP address, email, absolute file path
    static RE: Lazy<Regex> = Lazy::new(|| Regex::new(
        r"(\d{1,3}\.){3}\d{1,3}|[\w.]+@[\w.]+\.\w+|(/home|/Users|C:\\)[^\s]+"
    ).unwrap());
    RE.is_match(text)
}

fn kind_prior_score(kind: &str) -> f32 {
    match kind {
        "procedural" => 1.0,
        "fact"       => 0.9,
        "gotcha"     => 0.7,
        "decision"   => 0.6,
        "note"       => 0.3,
        "task"       => 0.0,
        "preference" => 0.0,
        _            => 0.3,
    }
}
```

```rust
pub async fn recompute_global_centroid(store: &dyn StoreBackend) -> Result<Vec<f32>> {
    let embeddings: Vec<Vec<f32>> = store.query(
        "SELECT embedding FROM memory WHERE scope_type = 'global' AND array::len(embedding) > 0"
    ).await?;
    if embeddings.is_empty() { return Ok(vec![]); }
    let dim = embeddings[0].len();
    let mut centroid = vec![0.0f32; dim];
    for emb in &embeddings {
        for (i, v) in emb.iter().enumerate() { centroid[i] += v; }
    }
    let n = embeddings.len() as f32;
    centroid.iter_mut().for_each(|v| *v /= n);
    Ok(centroid)
}
```

```rust
pub struct DocumentEngine {
    store: Arc<dyn StoreBackend>,
    embed: Arc<dyn Embedder>,
}

impl DocumentEngine {
    pub async fn ingest(
        &self,
        source: DocumentSource,
        project_id: Option<String>,
    ) -> Result<Document> {
        let (raw_text, title) = match &source {
            DocumentSource::File(p) => (std::fs::read_to_string(p)?, p.display().to_string()),
            DocumentSource::Text { content, title } => (content.clone(), title.clone()),
            DocumentSource::Url(_) => todo!("URL fetch in a later patch"),
        };

        // Create document record
        let doc_id = uuid::Uuid::new_v4().to_string();
        let doc = Document { id: doc_id.clone(), project_id: project_id.clone(),
            source: title.clone(), title: title.clone(), created_at: Utc::now() };
        self.store.query("INSERT INTO document $d").bind(("d", &doc)).await?;

        // Chunk with text-splitter (512 tokens, 64 overlap, markdown-aware)
        use text_splitter::{TextSplitter, ChunkConfig};
        let splitter = TextSplitter::new(ChunkConfig::new(512).with_overlap(64)?);
        let chunks: Vec<&str> = splitter.chunks(&raw_text).collect();

        let mut char_offset = 0;
        for chunk_text in chunks {
            let embedding = self.embed.embed(chunk_text).await?;
            let tok_count = chunk_text.split_whitespace().count(); // rough estimate
            self.store.query(
                "INSERT INTO chunk { document_id: $did, text: $t, embedding: $e, \
                 token_count: $tc, char_offset: $co }"
            )
            .bind(("did", &doc_id))
            .bind(("t", chunk_text))
            .bind(("e", embedding))
            .bind(("tc", tok_count))
            .bind(("co", char_offset))
            .await?;
            char_offset += chunk_text.len();
        }

        Ok(doc)
    }

    pub async fn search(
        &self,
        query: &str,
        project_id: Option<&str>,
        limit: usize,
    ) -> Result<Vec<ScoredChunk>> {
        let qvec = self.embed.embed(query).await?;
        let scope_filter = match project_id {
            Some(pid) => format!("AND document.project_id IN ['{pid}', NULL]"),
            None      => String::new(),
        };
        let results: Vec<ScoredChunk> = self.store.query(&format!(
            "SELECT chunk.*, \
             vector::similarity::cosine(chunk.embedding, $vec) AS score \
             FROM chunk \
             JOIN document ON chunk.document_id = document.id \
             WHERE 1=1 {scope_filter} \
             ORDER BY score DESC LIMIT {limit}"
        )).bind(("vec", qvec)).await?;
        Ok(results)
    }

    pub async fn list(&self, project_id: Option<&str>) -> Result<Vec<Document>> {

    }

    pub async fn delete(&self, doc_id: &str) -> Result<()> {
        self.store.query("DELETE chunk WHERE document_id = $did").bind(("did", doc_id)).await?;
        self.store.query("DELETE document WHERE id = $did").bind(("did", doc_id)).await?;

    }
}
```

### S7.1 — MMR in base recall (not just hybrid_search)

**File:** `crates/cairn-memory/src/lib.rs`

Currently MMR (Maximal Marginal Relevance) is only in `hybrid_search()`. Apply it to the
base `recall()` function too so all recall paths benefit from diversity:

```rust
fn mmr_rerank(results: Vec<ScoredMemory>, lambda: f32, k: usize) -> Vec<ScoredMemory> {
    // Standard MMR: iteratively pick the item that maximizes
    //   lambda * sim(query, item) - (1-lambda) * max_sim(item, already_selected)
    // lambda=0.7 = relevance-heavy; 0.3 = diversity-heavy
    // k = how many to return
    let mut selected: Vec<ScoredMemory> = Vec::new();
    let mut remaining = results;
    while selected.len() < k && !remaining.is_empty() {
        let best = remaining.iter().enumerate().max_by(|(_, a), (_, b)| {
            let a_rel = a.score;
            let a_div = selected.iter()
                .map(|s| cosine_similarity(&a.embedding, &s.embedding))
                .fold(0.0f32, f32::max);
            let a_mmr = lambda * a_rel - (1.0 - lambda) * a_div;
            let b_rel = b.score;
            let b_div = selected.iter()
                .map(|s| cosine_similarity(&b.embedding, &s.embedding))
                .fold(0.0f32, f32::max);
            let b_mmr = lambda * b_rel - (1.0 - lambda) * b_div;
            a_mmr.partial_cmp(&b_mmr).unwrap_or(Equal)
        }).map(|(i, _)| i);
        if let Some(i) = best {
            selected.push(remaining.remove(i));
        }
    }
    selected
}
```

Apply after RRF fusion with `lambda=0.7`, `k=limit`.
